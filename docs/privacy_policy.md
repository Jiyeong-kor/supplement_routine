# Supplement Routine 개인정보처리방침

시행일: 2026-06-05

Supplement Routine은 사용자가 직접 입력한 영양제 복용 루틴을 기기 안에서 관리하는 local-first 앱입니다. 이 앱은 영양제 추천, 의료 조언, 진단, 치료, 처방 기능을 제공하지 않습니다.

## 수집하는 개인정보

Supplement Routine은 서버 계정, 클라우드 동기화, 광고, 분석 SDK를 사용하지 않습니다. 앱은 사용자의 개인정보를 개발자 서버로 전송하거나 제3자에게 판매하지 않습니다.

앱 사용 중 사용자가 직접 입력할 수 있는 정보는 다음과 같습니다.

- 영양제 이름
- 복용량과 단위
- 복용 방식과 시간
- 복용 기록
- 메모
- 식사 시간 설정
- 알림 설정

이 정보는 앱 기능을 제공하기 위해 기기 로컬 저장소에 저장됩니다.

## 정보 이용 목적

앱은 사용자가 입력한 정보를 다음 목적으로만 사용합니다.

- 오늘 복용 일정 표시
- 복용 완료 여부 기록
- 최근 기록과 완료율 표시
- 사용자가 설정한 시간의 로컬 알림 예약
- 식사 시간과 알림 기본값 저장

## 제3자 제공

Supplement Routine은 사용자의 루틴 정보나 복용 기록을 제3자에게 제공하지 않습니다.

## 외부 전송

현재 Android 앱은 사용자의 루틴 데이터를 외부 서버로 전송하지 않습니다. 앱 데이터는 기기 안에 저장되며, 사용자가 직접 삭제할 수 있습니다.

## 권한 사용

앱은 다음 Android 권한을 사용할 수 있습니다.

| 권한 | 목적 |
| --- | --- |
| 알림 권한 | 사용자가 설정한 복용 알림을 표시하기 위해 사용합니다. |
| 정확한 알림 권한 | 사용자가 설정한 시간에 더 정확하게 알림을 예약하기 위해 사용합니다. 기기 설정 또는 OS 정책에 따라 부정확한 알림으로 대체될 수 있습니다. |

## 데이터 보관과 삭제

사용자가 입력한 데이터는 기기 로컬 저장소에 보관됩니다. 앱의 설정 화면에서 데이터 초기화를 실행하거나, 기기 설정에서 앱 데이터를 삭제하거나, 앱을 삭제하면 로컬 데이터가 삭제될 수 있습니다.

## 아동 개인정보

Supplement Routine은 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

## 정책 변경

개인정보처리방침이 변경되면 이 문서의 시행일과 내용을 갱신합니다.

## 문의

개인정보처리방침에 대한 문의는 Google Play 스토어 등록정보에 표시된 개발자 연락처를 통해 보낼 수 있습니다.
