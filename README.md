# Supplement Routine

사용자가 직접 입력한 영양제 복용 규칙을 기반으로  
복용 일정, 알림, 기록을 관리하는 Flutter 앱입니다.

## 프로젝트 목표

이 앱은 영양제 복용을 “추천”하지 않습니다.  
사용자가 이미 먹고 있는 영양제를 잊지 않고 꾸준히 복용할 수 있도록 돕는 것을 목표로 합니다.

## 핵심 방향

- 모든 복용 정보는 사용자 입력 기반입니다.
- 앱은 건강/의료 조언을 제공하지 않습니다.
- 복용 일정 생성과 기록 관리 기능에 집중합니다.

## MVP 기능

- 영양제 등록
- 오늘 복용 목록 확인
- 복용 체크
- 복용 기록
- 오늘의 한 줄 (메모 기반)

## 현재 진행 상태

- Flutter 프로젝트 초기 구조 생성
- feature 기반 폴더 구조 설계 완료
- 기본 앱 실행 상태 구성 완료
